# zingoHelper
Helper scripts for zebrad, zaino, and zingo-cli

## Reqs

Needs a fully synced zebrad, a compiled and running zaino or lightwalletd server, and a compiled zingo-cli.


## Examples

Once you loaded a seed phrase or viewkey and are fully synced, you can run exportToJSON.sh to export all transaction data to json.
